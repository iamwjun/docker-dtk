## docker-compose php + nginx environment

## 一、Common commands

1、boot image：
```
dokcer-compose up -d
```
2、down image：
```
dokcer-compose down
```
3、restart image：
```
dokcer-compose restart
```
4、build：
```
dokcer-compose build
```
4、update Dockerfile after build：
```
docker-compose up -d --build
```
